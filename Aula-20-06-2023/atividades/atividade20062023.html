Atividade - Tags de texto

Nesta atividade, você aprenderá a utilizar tags texto para formatar o conteúdo da pagina web

A seguir, temos um exemplo de trecho de código em HTML para enfatizar o texto

< em >Texto Enfatizado < /em > ;

temos outras opções de texto,como

< i > < em > < b > < strong > < u > < sub > < sup > , entre outras, para ver como elas alteram a aparência de texto.

podemos adicionar trecho de código usando < code > ... < /code >

var num = 2;
console.log(num);
